# 📊 K-Olive Demand & Customer Analytics

## 🔍 Overview
This project analyses customer behavior and demand patterns for K-Olive, a regional olive producer, using advanced statistical and predictive analytics techniques.

## 🧠 Key Techniques
- Multiple Linear Regression (Order Quantity)
- Interaction Effects (Quality × Brand Image)
- Logistic Regression (Customer Recommendation)
- Time-Series Forecasting (Quarterly Demand)

## 📈 Key Insights
- Product quality and social media presence are major drivers of order quantity.
- Interaction analysis shows quality impacts low brand perception customers more strongly.
- Customers purchasing directly are ~2.8x more likely to recommend.
- Strong seasonal demand pattern with peak in Q3.

## 🛠 Tools Used
- Excel (Regression, Forecasting)
- Statistical Modeling
- Data Visualization
